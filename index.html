Dataset Overview

MobileHCI'15: Userstudy 1: EyeTab Accuracy

This study assessed the accuracy of the EyeTab Computer Vision based gaze tracking approach.

MobileHCI'15: Userstudy 2: Use Case Evaluation

This user study concerns assessing the applicability of gaze tracking to real-life use cases. The first use case assesses the ability to detect if the user focuses on the device or gazes besides it. The second use case evaluates the ability to use mo- bile gaze tracking for supporting psychological field studies diagnosing dyslexia.
